## For devs

### Import & install the project in your IDE :
* open IDE's terminal and paste
  ```
  git clone git@github.com:Clean-Pick/React-Pokedex.git
  ```
* then
  ```
  npm install
  ```
* and finally
  ```
  npm run dev
  ```
